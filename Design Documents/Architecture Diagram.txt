PAYMENT FLOW:
User → Book → [Create Hold in DB] → [Create Stripe Session]
     → Redirect to Stripe Hosted Checkout
     → User pays → Stripe redirects to /payment-success?session_id=XXX
     → App calls stripe.checkout.Session.retrieve(session_id) server-side
     → If payment_status == 'paid' → confirm booking in DB
     → Else → show error, release hold

WEBHOOK (POST-MVP):
Stripe → [Separate FastAPI service on Railway/Render]
       → Verify signature → Confirm booking in Supabase
